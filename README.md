# Warehouse Digital Twin

Digital Twin of an Ozon warehouse sorting center built with Python and SimPy.

## Project Goal

Develop a discrete-event simulation of a warehouse sorting center that allows users to:

- simulate warehouse operations;
- analyze warehouse performance;
- identify bottlenecks;
- compare different warehouse configurations.

## Technology Stack

- Python
- SimPy
- Streamlit
- Plotly
- Pandas
- NumPy
- Docker
